# MNIST Dimensionality Reduction and Classification
This project demonstrates the application of dimensionality reduction techniques and classification algorithms on the MNIST dataset using Python and popular machine learning libraries such as Scikit-learn and TensorFlow.

## Dependencies
- numpy
- scikit-learn
- tensorflow

## Dataset
The project uses the **MNIST** handwritten digit dataset. There is also commented code for potential use of the **CIFAR-10** dataset.

## Functionality
The code performs the following:
1. Loads the MNIST dataset.
2. (Optional) Loads the CIFAR-10 dataset (currently commented).
3. Preprocesses the data.
4. Applies dimensionality reduction techniques such as:
   - Kernel PCA
   - Linear Discriminant Analysis (LDA)
5. Classifies the data using classifiers like:
   - Support Vector Machines (SVM)
   - K-Nearest Neighbors (KNN)
   - Nearest Centroid
