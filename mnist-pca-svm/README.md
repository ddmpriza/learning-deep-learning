# MNIST Classification with PCA and SVM

## Description
This project demonstrates the use of **dimensionality reduction** and **classification** on the **MNIST** dataset.  
The workflow includes:
- Preprocessing and normalization of image data,
- Applying **PCA (Principal Component Analysis)** to reduce dimensionality,
- Training an **SVM (Support Vector Machine)** classifier on the reduced feature set,
- Evaluating the classification performance.

There is also commented code for loading and experimenting with the **CIFAR-10** dataset.

---

## Requirements
To run the notebook, you need the following dependencies:

- Python 3.8+
- NumPy  
- Scikit-learn  
- TensorFlow
