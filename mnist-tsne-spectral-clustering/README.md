# MNIST Clustering with t-SNE and Spectral Clustering

## Description
This project applies **unsupervised learning** on the MNIST handwritten digits dataset using:

- t-SNE** (t-distributed Stochastic Neighbor Embedding) for dimensionality reduction (784 → 2),
- Spectral Clustering** for data grouping,
- 2D visualization** of the results.

The main goal is to explore how the digits are distributed in a reduced feature space and evaluate how well clustering can separate them.

## Requirements
To run the notebook, you need the following dependencies:

- Python 3.8+
- NumPy  
- Matplotlib  
- SciPy  
- scikit-learn  
- TensorFlow
